# AI Telegram Bot — YandexGPT Integration

Telegram-бот с интеграцией AI-модели (YandexGPT) через OpenAI-совместимый API. Бот ведёт диалог с пользователем, помнит контекст переписки и отвечает осмысленно, а не просто пересылает реплики.

## Возможности

- Диалог с AI-моделью (YandexGPT) прямо в Telegram
- Память контекста — бот помнит последние сообщения каждого пользователя отдельно
- Команда `/reset` — очистка истории диалога
- Обработка ошибок при недоступности AI-сервиса

## Стек

- **Python**
- **aiogram 3.x** — Telegram Bot API
- **OpenAI SDK** (используется как OpenAI-совместимый клиент для YandexGPT)
- **Yandex Cloud AI Studio** — языковая модель

## Архитектура

1. Пользователь отправляет сообщение боту
2. Бот добавляет сообщение в историю диалога этого пользователя
3. История + системный промпт отправляются в YandexGPT через API
4. Ответ модели возвращается пользователю и сохраняется в историю
5. История автоматически обрезается, чтобы не разрастаться бесконечно

## Запуск

1. Установить зависимости:
   ```
   pip install aiogram openai
   ```
2. Задать переменные окружения (или вписать напрямую в конфиг):
   - `BOT_TOKEN` — токен бота от @BotFather
   - `YANDEX_API_KEY` — API-ключ из Yandex Cloud AI Studio
   - `YANDEX_FOLDER_ID` — ID каталога в Yandex Cloud
3. Запустить:
   ```
   python ai_bot.py
   ```

## Возможные доработки

- RAG (поиск по базе знаний компании перед ответом модели)
- Function calling — вызов реальных функций (проверка остатков, статуса заказа и т.д.)
- Хранение истории в БД вместо памяти процесса (для устойчивости к перезапускам)
- Поддержка нескольких ролей/сценариев (например, бот поддержки vs бот-консультант)
